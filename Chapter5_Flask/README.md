рЃцрЃџрЃљрЃАрЃЎрЃў Python-рЃерЃў рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃЋрЃћрЃЉ рЃЏрЃўрЃЎрЃарЃЮ рЃцрЃарЃўрЃЏрЃЋрЃЮрЃарЃЎрЃўрЃљ. рЃърЃўрЃЌрЃЮрЃюрЃерЃў рЃЊрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃњрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃљрЃАрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃЊрЃљрЃ«рЃБрЃюрЃФрЃџрЃЮрЃЌ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃБрЃљрЃЏрЃарЃљрЃЋрЃў рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃўрЃЌ, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф [flask-рЃўрЃА рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃў рЃЊрЃљ third-party
рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў рЃњрЃЋрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃћрЃю](https://www.fullstackpython.com/flask-extensions-plug-ins-related-libraries.html).

рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ, рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃўрЃА рЃерЃћрЃЏрЃЮрЃбрЃљрЃюрЃљ:


```python
from flask import Flask
```

_рЃЌрЃБ рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃў рЃњрЃћрЃбрЃДрЃЮрЃЋрЃўрЃЌ рЃарЃЮрЃЏ flask рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ, рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃарЃЮрЃЏ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃАрЃљрЃцрЃћрЃ«рЃБрЃарЃў рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃљрЃа рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃљ._

## [рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃерЃљрЃЉрЃџрЃЮрЃюрЃў](https://flask.palletsprojects.com/en/1.1.x/quickstart/#a-minimal-application)

```python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "<h1>рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ</h1>"

app.run()
```

рЃЎрЃЮрЃЊрЃў | рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ
---|---
``from flask import Flask`` | рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ flask-рЃўрЃЊрЃљрЃю рЃерЃћрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћ Flask рЃЎрЃџрЃљрЃАрЃў
`app = Flask(__name__)` | рЃЦрЃЏрЃюрЃўрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА Flask-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ. рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃБрЃџрЃў ``__name__``-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃАрЃљрЃгрЃДрЃўрЃА рЃЏрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃљрЃА, рЃарЃўрЃА рЃЏрЃўрЃЏрЃљрЃарЃЌрЃљрЃф рЃћрЃФрЃћрЃЉрЃА/рЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃА.
`@app.route('/')` | [route рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃў](#route) рЃарЃЮрЃЏрЃћрЃџрЃўрЃф Fask-рЃА рЃљрЃгрЃЋрЃЊрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃЌрЃБ рЃарЃљ URL-рЃќрЃћ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃЊрЃћрЃА рЃЏрЃўрЃА рЃЦрЃЋрЃћрЃе рЃЏрЃЊрЃњрЃЮрЃЏрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ.
```def home(): return "<h1>рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ</h1>" ``` | рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЦрЃбрЃўрЃБрЃарЃЊрЃћрЃЉрЃљ рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃўрЃЌ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ, рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃњрЃљрЃгрЃћрЃарЃўрЃџ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃА рЃЊрЃљ return-рЃўрЃЌ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃћрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА.
`app.run()` | app рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА run рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ рЃЕрЃЋрЃћрЃюрЃА рЃЏрЃўрЃћрЃа рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃў Flask рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ. (рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃўрЃАрЃљрЃА рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЊрЃљрЃАрЃфрЃћрЃЌ port рЃфрЃЋрЃџрЃљрЃўрЃА рЃЏрЃюрЃўрЃерЃюрЃЋрЃћрЃџрЃЮрЃЉрЃљ рЃарЃљрЃф рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџ рЃърЃЮрЃарЃбрЃќрЃћ рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ)
